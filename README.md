# ss-music-plugins

个人 [MusicFree](https://github.com/maotoumao/MusicFree) 音源插件仓库。

## 订阅地址

在 MusicFree 中 `设置 → 插件管理 → 安装插件 → 从网络安装`,粘贴:

```
https://raw.githubusercontent.com/sharpHL/ss-music-plugins/main/plugins.json
```

## 当前包含

| 插件 | 文件 | 来源 |
|---|---|---|
| 网易云音乐 | `plugins/netease.js` | Fork 自 [ThomasBy2025/musicfree](https://github.com/ThomasBy2025/musicfree) |
| 小宇宙 FM | `plugins/xiaoyuzhou.js` | 自写,基于 [ultrazg/xyz](https://github.com/ultrazg/xyz) 整理的 API |

## 使用约束

仅供个人学习和自用。不要公开分发本仓库的派生 .apk,不要拿来商用。
